# EMall商城

## 一、项目技术栈

- 前端：Vue，Ajax
- 后端：SpringBoot、MybatisPlus
- 数据库：MySQL
- 认证鉴权：Shiro、Jwt

## 二、项目部署

1. 新建数据库：emall
2. 导入数据库文件：emall.sql
3. 加载meven工程
4. 更改项目数据库配置
5. 启动项目

## 三、项目相关地址

- 项目地址：http://localhost:12001/

- Swagger文档地址：http://localhost:12001/swagger-ui/
- 项目前端地址: https://github.com/psikun/Emall-admin-web
